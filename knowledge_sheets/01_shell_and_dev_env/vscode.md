# VS Code — Common Keyboard Shortcuts (macOS Cheat Sheet)

This cheat sheet lists the **most common and useful VS Code shortcuts**
for fast daily work (data science, Python, engineering).

---

## Command Palette (most important)

```text
Cmd + Shift + P
```

Open the Command Palette (run any command, change interpreter, settings, etc.).

---

## Python Interpreter

```text
Cmd + Shift + P → Python: Select Interpreter
```

Select or change the active Python environment.

---

## Multi-cursor / Multi-edit

```text
Option + Click
```

Add an extra cursor.

```text
Option + Shift + Click
```

Add multiple cursors vertically.

```text
Cmd + Option + ↑ / ↓
```

Add cursor above / below.

```text
Cmd + D
```

Select next occurrence of the current word.

---

## File Navigation

```text
Cmd + P
```

Quickly open a file by name.

```text
Cmd + Shift + O
```

Go to a symbol in the current file.

```text
Cmd + T
```

Go to a symbol in the workspace.

---

## Search & Replace

```text
Cmd + F
```

Find in current file.

```text
Cmd + Option + F
```

Replace in current file.

```text
Cmd + Shift + F
```

Search across the workspace.

```text
Cmd + Shift + H
```

Replace across the workspace.

---

## Refactor & Code Actions

```text
Cmd + Shift + R
```

Refactor symbol (when supported).

```text
Cmd + .
```

Quick Fix / Code Actions.

---

## Line Editing

```text
Cmd + /
```

Toggle line comment.

```text
Option + ↑ / ↓
```

Move line up / down.

```text
Cmd + Shift + K
```

Delete line.

```text
Cmd + Enter
```

Insert line below.

```text
Cmd + Shift + Enter
```

Insert line above.

---

## Formatting

```text
Option + Shift + F
```

Format document.

```text
Cmd + K Cmd + F
```

Format selection.

---

## Tabs & Editor Groups

```text
Cmd + W
```

Close current tab.

```text
Cmd + Shift + [
Cmd + Shift + ]
```

Switch between editor tabs.

```text
Cmd + \
```

Split editor.

---

## Terminal

```text
Ctrl + `
```

Toggle integrated terminal.

```text
Cmd + Shift + `
```

Create new terminal.

---

## Debugging

```text
F5
```

Start / continue debugging.

```text
Shift + F5
```

Stop debugging.

```text
F9
```

Toggle breakpoint.

---

## Zen Mode (focus)

```text
Cmd + K Z
```

Enter / exit Zen Mode.

---

## Notes

- Most commands are accessible via **Cmd + Shift + P**
- Learn multi-cursor → massive productivity gain
- Refactor shortcuts depend on language support
